# Hearts 0.9b
The card game Hearts for QT (linux)

1. Requirement:
- Qt Creator
- QT Widget 5
- Liballegro 5

2. Install: (under ubuntu 16.04)
- sudo apt-get update
- sudo apt-get install liballegro5-dev
